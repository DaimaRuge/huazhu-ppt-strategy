# eduClaw - 儿童学习系统

基于 OpenClaw 的儿童个性化学习管家系统。

## 项目简介

eduClaw 是一个基于 AI Agent 的儿童学习系统，核心思想是将 OpenClaw 作为"学习管家"，而不是像 ChatGPT 那样的"解题高手"。OpenClaw 可以按设定的规则主动运转，沉淀数据。

## 产品特点

- 🤖 **AI 学习管家**：OpenClaw 按规则主动运转，而非被动问答
- 🔒 **账号隔离**：家长和孩子各有独立频道，权限分级
- ✨ **积分系统**：永不扣分，让努力被看见
- 📊 **数据沉淀**：从"感觉进步"到"看得见的进步"
- 🎮 **游戏化体验**：单题模式减轻心理负担

## 核心功能

### 英语学习
- 单词造句练习（8-12词）
- 三维度评分（内容/词汇/语法）
- IELTS 风格口语问答（儿童适配版）
- 单词形态归并去重

### 数学学习
- 单题模式，减轻心理负担
- 自适应难度
- 错题自动进复习池

### 积分系统
- 完成练习 +5 分
- 全对额外 +3 分
- 连续学习第 N 天额外 +N 分（上限 +7）
- 永不扣分

## 快速开始

### 前置条件
- OpenClaw 运行环境
- 飞书企业账号（用于账号隔离）
- iPad（推荐儿童端使用）

### 部署步骤
1. 初始化 OpenClaw 工作区 `workspace-grace`
2. 配置 `SOUL.md` 和 `USER.md`
3. 开发 `primary-homework-coach` Skill
4. 配置飞书账号隔离
5. 创建飞书多维表格
6. 测试端到端流程

## 关键技术点

- **单词形态归并**：规则匹配+词形还原
- **三维度评分**：内容+词汇+语法
- **账号隔离**：飞书多账号+权限路由
- **数据沉淀**：飞书云文档+多维表格
- **性能优化**：精简工具 schema，只保留聊天功能
