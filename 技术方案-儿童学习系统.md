### 4.2 积分系统

```python
class PointsSystem:
    def __init__(self, user_id):
        self.user_id = user_id
        self.total_points = 0
        self.consecutive_days = 0
    
    def complete_exercise(self, is_correct):
        points = 5  # 基础分
        
        if is_correct:
            points += 3  # 全对额外分
        
        # 连续学习奖励
        points += min(self.consecutive_days, 7)
        
        self.total_points += points
        return {
            "earned": points,
            "total": self.total_points,
            "consecutive_days": self.consecutive_days,
            "message": f"本题 +{points} 分，累计 {self.total_points} 分"
        }
```

### 4.3 单词形态归并

```python
# 词形归一化
def lemmatize_word(word):
    # 规则库
    irregular = {
        "received": "receive",
        "chuckled": "chuckle",
        "staring": "stare"
    }
    
    # 规则匹配
    if word in irregular:
        return irregular[word]
    
    # 后缀去除
    suffixes = ["ed", "ing", "s", "es"]
    for suffix in suffixes:
        if word.endswith(suffix):
            return word[:-len(suffix)]
    
    return word

# 去重入库
def add_word_to_library(word):
    lemma = lemmatize_word(word)
    if lemma not in word_library:
        word_library[lemma] = {
            "lemma": lemma,
            "forms": [word],
            "difficulty": infer_difficulty(lemma)
        }
    else:
        if word not in word_library[lemma]["forms"]:
            word_library[lemma]["forms"].append(word)
```

## 6. 性能优化

### 6.1 飞书插件优化

**问题**：飞书插件响应极慢

**根因**：工具 schema 定义过于庞大

**解决方案**：
```yaml
# channels.feishu.tools 配置
channels:
  feishu:
    tools:
      doc: false      # 不需要直接调用文档API
      chat: true      # 只保留聊天
      wiki: false
      drive: false
      perm: false
      bitable: false  # 不需要直接调用多维表格API
```

### 6.2 响应优化策略
- 只加载必要的工具 schema
- 异步数据回填
- 缓存常用数据

### 7.2 权限控制

```python
class PermissionManager:
    def check_permission(self, user_id, action):
        user_role = get_user_role(user_id)
        
        permissions = {
            "admin": [
                "view_all_data",
                "adjust_rules",
                "modify_plan",
                "view_feedback"
            ],
            "child": [
                "do_exercise",
                "submit_homework",
                "view_own_feedback",
                "view_points"
            ]
        }
        
        return action in permissions.get(user_role, [])
```
